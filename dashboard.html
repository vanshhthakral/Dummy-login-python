Project Notes

This project demonstrates a fully integrated DevSecOps pipeline that includes:

  1. Code is pushed to GitHub for version control.
  2. Jenkins triggers the CI/CD pipeline and initiates security scanning.
  3. High-severity vulnerabilities block further stages.
  4. Only secure and validated builds proceed to Docker image creation and deployment.
